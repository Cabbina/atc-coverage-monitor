Per implementare una nuova feature:
1. Crea un branch git: feature/nome-feature
2. Aggiorna il tipo ATCPosition in normalizer.js se necessario
3. Scrivi i test unitari prima del codice (TDD)
4. Aggiorna il README se l'interfaccia pubblica cambia
5. Crea una PR con descrizione della feature e screenshot/test output
